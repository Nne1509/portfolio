PREDICTIVE MODELLING FOR AGRICULTURE

The data, found here shows the crops in a farm and the distribution of soil minerals for each of the crops. This repository contains the code and resources for a machine learning model that helps a farmer to decide the best crops to plant in his field based on the condition of the soil. This model analyzes soil metrics such as Nitogen, Potassium, Phosphorous and pH levels of the soil to assist the farmer in making the best decision.

The aim of this project was to build a multi-class classification model to predict suitable crops for the farmer. We employed logistic regression to achieve this and plotted a heat map to guide our feature selection in order to avoid multicollinearity.

The data exploration for this project consisted of the following:

  • Handling missing values
  • Crop Identification
  • Feature Correlation

Tools used: Python (pandas, numpy, matplotlib, seaborn,scikit-learn)

INVESTIGATING NETFLIX MOVIES

The data, found here contains data about movies and shows on netflix, separated the relevant columns and filtered them to determine the movies shorter than 60 minutes and inspected the results to find contributing factors.

The data exploration for this project consisted of the following:

  • Loading and inspecting the dataset
  • Filtering the data to remove TV shows as we are focusing on movies
  • Creating a subset called netflix_movies including only the columns: title, country, genre, release_year and duration
  • Filtering the netflix_movies subset to find movies that are shorter than 60 minutes and saving it as short_movies
  • Creating a dictionary to assign colors to genres to add detail and aid readability of the scatter plot
  • Create a scatter plot for movie duration by release year
  • Inspect scatter plot to determine if we are certain that movies are getting shorter as the years go by

Tools used: Python (pandas, matplotlib)
